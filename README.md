-------
Every table that users can edit should contain these columns:

UPDATED_AT
SYNC_STATUS

where

0 = Synced
1 = Pending Upload
2 = Upload Failed

Whenever your application inserts or updates locally:

UPDATED_AT = CURRENT_TIMESTAMP
SYNC_STATUS = 1

After Oracle accepts the change:

SYNC_STATUS = 0
